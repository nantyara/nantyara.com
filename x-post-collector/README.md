# X Post Collector - Chrome拡張

X（旧Twitter）のアカウントから投稿を自動収集するChrome拡張機能です。

## 機能

- 自動スクロールで過去の投稿を全て収集
- 投稿テキスト、画像URL、日時、エンゲージメント数を抽出
- JSON形式でエクスポート
- 画像を一括ダウンロード
- 複数アカウントに対応

## インストール方法

1. Chromeで `chrome://extensions/` を開く
2. 右上の「デベロッパーモード」を有効にする
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. この`x-post-collector`フォルダを選択

## 使い方

1. Xでプロフィールページを開く
   - 例: https://x.com/nantyaraidol
2. 拡張機能アイコンをクリック
3. 「収集開始」ボタンを押す
4. 自動でスクロールしながらデータ収集（数分かかる場合あり）
5. 収集完了後、「JSONダウンロード」でデータをエクスポート
6. 必要に応じて「画像一括ダウンロード」で画像取得

    {

## 注意事項

- スクレイピングはXの利用規約でグレーゾーンです。個人利用の範囲で使用してください
- 大量のリクエストはアカウント制限の可能性があります
- データ収集には時間がかかる場合があります（投稿数による）
- 画像ダウンロードは順次行われるため時間がかかります

## アイコンについて

`icon.png`が必要です。128x128pxの画像を用意してください。
仮のアイコンがない場合、拡張機能は動作しますが警告が出ます。

## トラブルシューティング

### データが収集されない
- Xのプロフィールページ（`/ユーザー名`）で実行していますか？
- ページが完全に読み込まれてから実行してください
- F12で開発者ツールを開いてコンソールログを確認

### 画像がダウンロードされない
- Chrome の設定でダウンロード許可が必要です
- ダウンロードフォルダの容量を確認してください

### 拡張機能が読み込めない
- manifest.jsonの構文エラーがないか確認
- Chrome拡張のバージョンがManifest V3に対応しているか確認
